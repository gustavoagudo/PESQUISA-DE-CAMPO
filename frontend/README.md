# Formulário de Pesquisa Acadêmica - FAI

Formulário de pesquisa sobre percepções e motivações por trás do uso de antidiabéticos por indivíduos não diabéticos na busca por emagrecimento.

## Instalação

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev

# Build para produção
npm run build

# Visualizar build de produção
npm run preview
```

## Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- Radix UI
- Lucide Icons

## Estrutura

- `src/App.tsx` - Componente principal com gerenciamento de estado
- `src/components/consent-page.tsx` - Página de consentimento (TCLE)
- `src/components/data-collection-page.tsx` - Página de coleta de dados pessoais
- `src/components/final-data-page.tsx` - Página final com questões sobre medicamentos
- `src/components/success-page.tsx` - Página de confirmação
